# Process Management

## What is a Process?

A process is a running instance of a program.

Example:

Program:
firefox

Process:
Firefox running in memory.

---

## PID

PID stands for Process ID.

Each running process has a unique identifier.

Example:

Firefox -> PID 4521

---

## ps

Used to display running processes.

Examples:

ps

ps aux

---

## top

Displays system activity in real time.

Shows:

- CPU usage
- RAM usage
- Running processes

---

## kill

Used to send signals to processes.

Examples:

kill PID

kill -9 PID

---

## Difference Between Program and Process

Program:
Stored on disk.

Process:
Running in memory.
